Para poder realizar la detección de objetos es necesario yolov3.weights

Debido al gran tamaño de este archivo hemos decidido poner el enlace de descarga y como ponerlo en el codigo para poder hacer la simulación en python del detector de objetos

1-Ve al siguiente enlace: https://pjreddie.com/darknet/yolo/

2-Dirigete a la sección Performance on the COCO Dataset dentro de la pagina

3-Descarga el YOLOv3-608 weights (En el caso de dar cualquier problema descarguese también el .cfg y elimine el del proyecto actual)

4-Ponga el Yolov3.weights en esta misma carpeta YOLOv3
